# AeroNexus Frontend

## Overview

The AeroNexus Frontend is a React-based web application that provides a modern interface for managing flights, aircraft, operational records, and user accounts within the AeroNexus Flight Management System.

## Features

* User Login and Registration
* Dashboard Interface
* Aircraft Management
* Flight Management
* Flight Records Management
* Responsive User Interface
* Backend API Integration

## Technologies Used

* React
* Vite
* Tailwind CSS
* React Router DOM
* JavaScript

## Requirements

Before running the frontend, install:

* Node.js
* npm

Verify installation:

```bash
node -v
npm -v
```

## Installation

Clone the repository:

```bash
git clone https://github.com/2540007-cmd/AeroNexus---Flight-Management-System.git
```

Navigate to the frontend folder:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

## Run the Development Server

```bash
npm run dev
```

After the server starts, open the URL displayed in the terminal (usually http://localhost:5173).

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Connecting to the Backend

Make sure the backend server is running before starting the frontend.

Backend:

```bash
.\server.exe
```

Frontend:

```bash
npm run dev
```
